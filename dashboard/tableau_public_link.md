# Dashboard no Tableau Public

Este dashboard apresenta uma visão executiva de vendas de bebidas, utilizando métricas estimadas de receita, margem e volume.

## Link público

[Acessar dashboard no Tableau Public](https://public.tableau.com/app/profile/daniel.de.camargo7724/viz/BeverageSalesAnalyticsDashboard/BeverageSalesAnalyticsDashboard)

## Métricas disponíveis

- Receita estimada
- Margem estimada
- Margem percentual estimada
- Volume total
- Receita por marca
- Margem por região
- Receita por canal
- Volume por pacote

## Filtros disponíveis

- Período
- Status do Volume
- Marca
- Região
- Grupo do Canal
- Tipo do Canal
- Pacote

## Observação

Os valores financeiros são estimados para fins de portfólio profissional e não representam dados reais de negócio.
