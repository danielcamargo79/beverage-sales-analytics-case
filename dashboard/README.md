# Dashboard

Esta pasta concentra os materiais relacionados à proposta de visualização do case de vendas de bebidas.

## Objetivo

Documentar a proposta de dashboard executivo e analítico para acompanhamento da performance comercial.

## Estrutura prevista

```text
dashboard/
├── mockups/
└── screenshots/
```

## Mockups

A pasta `mockups/` será utilizada para armazenar desenhos, protótipos ou wireframes do dashboard.

## Screenshots

A pasta `screenshots/` será utilizada para armazenar imagens da versão final ou parcial do dashboard.

## Ferramentas sugeridas

* Power BI Desktop
* Tableau Public
* Looker Studio
* Figma
* Excalidraw
* Canva

## Visões sugeridas

* Visão executiva
* Análise por produto
* Análise por região
* Análise por canal
* Evolução temporal de vendas
* Ranking de produtos
