# Decisões de DataViz

Este documento descreve as principais decisões de visualização propostas para o dashboard de análise de vendas de bebidas.

## Objetivo do dashboard

Criar uma visão executiva e analítica que permita acompanhar a performance comercial, identificar oportunidades e apoiar decisões de negócio.

## Público-alvo

* Diretoria comercial
* Gerentes regionais
* Time de vendas
* Business Analysts
* Data Analysts
* Área de BI/DataViz

## Estrutura sugerida do dashboard

### 1. Visão executiva

Indicadores principais no topo da página:

* Receita líquida
* Quantidade de pedidos
* Ticket médio
* Quantidade de itens vendidos
* Crescimento percentual

### 2. Análise temporal

Gráfico de linha para acompanhar a evolução das vendas ao longo do tempo.

Objetivo:

* Identificar tendência
* Detectar sazonalidade
* Comparar períodos

### 3. Ranking de produtos

Gráfico de barras com os produtos de maior receita ou maior volume vendido.

Objetivo:

* Identificar produtos líderes
* Avaliar concentração de vendas
* Apoiar decisões comerciais

### 4. Receita por categoria

Gráfico de barras ou treemap para comparar categorias de bebidas.

Objetivo:

* Entender representatividade por grupo
* Identificar categorias mais relevantes

### 5. Receita por região

Mapa ou gráfico de barras por região.

Objetivo:

* Avaliar desempenho geográfico
* Identificar regiões com potencial de crescimento

### 6. Receita por canal

Gráfico de barras ou coluna para comparar canais de venda.

Objetivo:

* Entender quais canais concentram maior receita
* Apoiar decisões de estratégia comercial

## Filtros sugeridos

* Período
* Região
* Canal
* Categoria
* Produto

## Boas práticas aplicadas

* Priorizar clareza sobre complexidade visual.
* Usar KPIs no topo para leitura rápida.
* Separar visão executiva de análise detalhada.
* Evitar excesso de cores.
* Destacar tendências e variações relevantes.
* Manter filtros simples e úteis.
* Construir o dashboard orientado a perguntas de negócio.
